terminal — how to ⌘K

Everything in the corpus is an entity — a person, a forum, an evidence signal, a run, a document. Every page is an entity × function. One command line reaches all of it: press ⌘K (or /), type a few letters of what you want, optionally name a function, press enter. No menus, no paths — the whole corpus behind one prompt.

the grammar

The line parses right-to-left: if the last word is a function mnemonic, it is the function, and everything before it is the entity query. Three shapes:

  • entity FUNCspolsky pre — fuzzy-match an entity, then open a named function on it.
  • entitygsig-008 — no function named, so the entity's default page opens (the first in its function list).
  • FUNCnet, help — a bare function; the two global commands need no entity at all.

Matching is fuzzy and forgiving — an fzf-style subsequence score over each entity's searchable text, with bonuses for hits at word boundaries and in an unbroken run, small penalties for scattered gaps. joel, spolsky, and jspol all land on the same seat. Naming a function first narrows the pool to the entity types that accept it — so a line ending in adj only ever ranks forums.

functions mnemonic · what it opens · types
  • DESdescribe — the entity profile page.person · forum · signal · doc
  • PREblind pre-interview document, written before the debate.person
  • POSTpost-debate interview document.person
  • DEBmoderated debate transcript, one turn at a time.forum
  • ADJtyped adjudication — converged actions and residual fork.forum
  • NETnetwork graph — global, or centred on the named entity.person · forum · global
  • OPENleave the terminal, open the run in the viewer.run
  • HELPthe built-in grammar reference, rendered inline.global

A type's default is the first function it lists: person → DES, forum → DES, run → OPEN. NET as a function centres the graph on the entity — kleppmann net opens the room around that seat; bare net opens the whole room.

keys
  • ⌘K / Ctrl-Kopen or close the palette from anywhere on the page.
  • /open the palette (when the cursor isn't already in a text field).
  • ↑ ↓move the selection through the ranked rows.

  • escclose the palette.
  • 1 – 7jump to a view — 1 overview · 2 screen · 3 triage · 4 forum · 5 axes · 6 memo · 7 network.
six lines, six pages each resolves against today's corpus

Forum seats are AI-simulated personas of public figures, clearly labelled as such in the corpus.

  • spolsky preJoel Spolsky's blind pre-interview — his opening read on the crux, written from his seat's mandate alone, before anyone argued.
  • gbrain adjthe g-brain forum's typed adjudication — converged actions, the residual fork routed to the human, and who moved versus who held.
  • gsig-008an evidence signal (default DES) — the dual-engine-parity finding, and its cited-by list: Spolsky, Gabriel, Kleppmann.
  • ferrite-inverted openleaves the terminal and reloads the run viewer on the ferrite-inverted tape.
  • netthe whole room as a force graph — forums, seated and suggested attendants, and the signals their mandates cite.
  • helpthe grammar reference rendered inline — the same mnemonics, one keystroke away when you forget one.
the citation spine

Nothing in the terminal is a dead end. Every gsig-* chip and every person's name is a link — on interview pages, in the debate transcript, throughout the adjudication. Click a signal chip and you land on its page; each signal carries a cited-by list naming the seats whose mandate leans on it, and each of those names links back to the person. The net graph draws the same relations in space: forums hold their attendants, attendants reach the evidence they cite. The corpus is one graph — the terminal is three ways of walking it.

the second register

Everything above is the first register: a human types the grammar and reads the page. The planned second register is an agent that drives the same grammar visibly — it opens spolsky pre, follows a gsig-* citation to its cited-by seats, pivots to net, and every move it makes is a line you could have typed yourself. Parity is the whole point: no hidden verbs, no private API — the agent's transcript reads back as terminal history.
